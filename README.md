# Logistics-Sales-Delivery-Performance-Analytics

1. Giới thiệu dự án

Dự án tập trung phân tích hiệu suất kinh doanh trong lĩnh vực logistics dựa trên dữ liệu bán hàng, khách hàng và sản phẩm.

Mục tiêu:

Theo dõi các chỉ số quan trọng: Revenue, Orders, AOV

Phân tích hành vi khách hàng bằng RFM

Đánh giá hiệu suất sản phẩm

Phân tích xu hướng doanh thu theo thời gian

Xây dựng dashboard trực quan bằng Excel

Tạo business insight phục vụ ra quyết định

Quy trình thực hiện:

SQL → Python → Excel Dashboard → Business Insight
